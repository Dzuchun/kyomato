## Блоки коду

Наразі мають досить обмежену підтримку, адже у $\LaTeX$ немає нативного поняття блоків коду.

На щастя, у **markdown** таке поняття є[^code]:
```rust
let x = 5;
let y = x.powi(2);
let z = (y as f64).sqrt();
println!("x = {x}, y = {y}, z = {x}");
```

[^code]: Принаймні, *Obsidian* таке робить
